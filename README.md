 LUMIÈRE — Luxury Makeup Brand

> *Where Beauty Meets Elegance*

A full-stack luxury makeup brand web portal built as a semester 
project for Web Engineering.

# Built With
- HTML5, CSS3, JavaScript
- PHP 8
- MySQL
- XAMPP (local server)

 Features
- User registration & login system
- Product shop with category filtering (AJAX)
- Shopping cart & checkout
- Beauty blog with comments
- Community glam feed with likes
- Full admin dashboard

How to Run Locally
1. Install XAMPP
2. Clone this repo into `htdocs/lumiere`
3. Import `lumiere.sql` into phpMyAdmin
4. Visit `http://localhost/lumiere`
